# 55-自动化文字叠加（2025年8月25日新增）

## Page 1

使用 XnConvert（一个免费的批量图片转换器），你可以轻松编辑图片甚至自动化文字叠加。
以下是我喜欢的方式（PS：我在这里混合并重新制作了 Pin，使用的是 Ideogram——原始 Pin 不是我的）：

1. 将你的 Pin 拖入输入（它们将被自动处理）。
2. 在操作下，随意编辑 Pin（你可以根据需要更改/扩展/调整）。
