# 59-使用 Ideogram 创建模板（2025年10月25日新增）

## Page 1

**结果：**
(1) 发现当前排名的内容，(2) 使用 Ideogram 克隆设计逻辑，用它创建模板，(3) 快速为你自己的关键词生成使用该设计的高表现变体。

正如你现在所知道的，我建议你在创建新 Pin 之前，快速查看（隐身标签页）该关键词当前哪些 Pin 在排名。
这样，你可以创建类似且更有可能表现良好的 Pin。

在此过程中，你经常会遇到你以前没有使用过的独特或新的 Pin 设计——定期创建和上传新鲜设计始终是一个好主意。
当你找到你喜欢的设计时，你可以轻松地将这些设计转变为模板。

**开始之前（2 分钟）**
使用一个新的隐身（私人）窗口。
准备一份你想要 Pin 的目标关键词简短列表（例如，"舒适圣诞卧室"、"20 美元以下极简圣诞礼物"、"冬季胶囊衣橱"）。

**步骤 1 - 找到正在排名的设计（隐身）**

1. 在隐身标签页中打开 Pinterest。
2. 搜索你的其中一个目标关键词。

### 3. 滚动并记录 3-5 个满足以下条件的 Pin：

清楚地获得了分发（在顶部/靠近顶部，显示保存/参与），具有你可以重用的设计风格（布局、类型层次、色块）。

4. 将它们的图片保存到你的计算机（仅供参考）。
   我们正在复制风格，而不是另一个创作者的内容。

## Page 2

**步骤 2 - 将排名的 Pin 转变为可重用的模板（Ideogram 混音）**

1. 将保存的图片之一拖放到 Ideogram 中。
2. 点击混音。
   Ideogram 将自动生成一个提示词，该提示词重新创建构图、排版、间距等。

## Page 3

### 3. 仔细检查生成的提示词文本：

如果原始 Pin 在设计中有可见的 URL（例如，"www.yourdomain.com"），请将该 URL 替换为你自己的（或将其移除）。
如果没有 URL，你可以添加一个简单的 URL 以进行品牌保护（可选）。

4. 如果需要，调整小细节（字体色调、间距词如 "&" 与 "and"），但保持核心结构——这就是排名的原因。
   这个提示词是你在这个确切设计风格上的"主模板"。

**步骤 3 - 为你所有的关键词（在那个主题中）倍增那个模板**

你现在将创建一批提示词变体，它们保持确切的设计，但根据关键词调整标题文本、颜色和微风格。

输出：为每个关键词返回一个完整的 Ideogram 提示词，每个占一行，保持与基础提示词相同的结构。

**结果：**
ChatGPT 返回一个整洁的即用提示词列表，每个针对不同的关键词调整，但具有已经排名的完全相同的布局/格式。
💡 当你的关键词在同一个主题内（例如，都是圣诞节）时，这效果最好，但作为其他细分领域的一般设计模板也非常棒。

当然，你也可以将这些提示词粘贴到 Ideogram 批量工作表中以自动化它们的生成。

1. 在 Ideogram 中，打开风格 → 创建新风格。
2. 添加你下载的 Pin（可选地添加该系列的 1-2 个更多）。
