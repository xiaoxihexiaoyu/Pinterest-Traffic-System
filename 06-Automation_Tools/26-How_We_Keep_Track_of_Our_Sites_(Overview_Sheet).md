# 内容

### Overview sheet 模板：

https://docs.google.com/spreadsheets/d/1YJVTyhN6NsZhf4OgrDXgJURGPRq7IvB_XJgxnI8W8rY/copy

### 我们在每个网站的 Overview sheet 中组织每个网站。

因此，我们添加关键词、文章 URL、标题、Pin 和 Board 描述。我们还会检查文章的 Pin 是否已上传以及文章是否完成。

### 词语含义：

关键词（Board）：在这里添加你想用你的网站定位的所有关键词。

相关文章：在这里添加已完成文章的 URL。

标题：在这里输入博客文章的吸引人的标题。

### 描述：这是 Pin 描述。

已创建文章？：帮助你（或你的团队）跟踪文章是否已创建。

已上传 Pin？：一旦这篇文章的所有 Pin 都已上传，你就可以勾选此项。

或者，你可以在 Pin 已创建并上传到 Google Drive 时勾选此项。
我们通常使用第二个选项，这样我们可以快速看到我们需要在哪里创建新 Pin。

Board 描述：这是你要上传此关键词 Pin 的 Board 的描述。

### 这是我们的简单工作流程：

当我向 Sheet 添加关键词时，我的商业伙伴通常会写文章，我将 Pin 上传到我们的 Google Drive 文件夹（每个关键词都有自己的 Pinterest Board 和 Google Drive 文件夹；名称完全匹配）。

### 示例：

关键词（添加到 Overview Sheet）： "Boho Living Room Decor"

### Google Drive 文件夹："Boho Living Room Decor"

然后我们将这些 Pin 上传到 Pinterest Board "Boho Living Room Decor"

### 有道理吗？

我通常一次性完成关键词研究，一次添加至少 20-30 个新关键词。
紧接着，我创建标题，如 Pin 和 Board 描述（使用我们的生成器）。

我喜欢尽可能简单地将这些任务在一个批次中完成 - 但当然，这取决于你和什么对你最有效。

这也意味着写文章的人不需要想出吸引人的标题 - 他们只需使用 Overview sheet 中的标题。

### 为什么我们使用 Overview Sheet 来组织我们的网站

使用这个系统，我们始终知道网站的状况 - 它有多少文章，我们已经为哪些关键词上传了 Pin，以及哪里需要更新（新 Pin、新文章、标题等）。

它也使得调整标题或 Pin 描述变得容易，特别是当多次链接到同一文章时。（我建议每向同一 URL 上传 10-15 个 Pin 后更新它们以保持新鲜！）

### 我们的 Overview Sheet 工作流程概览：

1. 将关键词添加到 Overview sheet

2. 为每个关键词添加标题、Board 和 Pin 描述（使用我们的生成器自动完成此操作）
3. 创建文章并使用现有标题，并在上线后勾选
4. 为这些文章创建 Pin 并将其上传到它们的 Google Drive 文件夹（将文件夹命名为关键词）
5. 将 Pin 上传到 Pinterest。

在下一课中，我将展示如何在 Google Drive 中组织我们的文件夹，以确保每个关键词都有自己的专用文件夹。
