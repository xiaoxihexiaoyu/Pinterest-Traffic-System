想要自动运行吗？

不再需要手动点击。你可以安排 Automation 自动清理 Pin，甚至在你睡觉时。

方法如下：

### 在 Make.com 中设置调度

1. 打开你的场景
2. 点击底部工具栏中的时钟图标
3. 启用调度
4. 选择你想要的频率：
   - 每天
   - 每周
   - 每 X 小时
   - 自定义（例如，每周日凌晨 4 点）

最佳实践：
每月运行一次清理，作为你内容卫生程序的一部分，例如每月 1 号。

### 你能"过度清理"吗？可以。

虽然 Pinterest 不会因为删除低表现 Pin 而惩罚你，但每次 API 调用都会被记录。过度使用 Automation，即使是删除，也可能触发垃圾邮件信号。

特别是如果：
- 你每小时删除数千个 Pin
- Automation 运行太频繁而没有休息
- 你正在并行运行多个自动化操作

经验法则：
- 持续清理，但不要激进
- 优先考虑质量过滤器而不是数量
- 监控平均有多少 Pin 符合你的过滤器条件 - 并相应调整调度

根据我们的经验，这从未导致封号。

相反，它通常会快速提升参与度和保存率，通常只需几天，新 Pin 也开始更快获得展示。
