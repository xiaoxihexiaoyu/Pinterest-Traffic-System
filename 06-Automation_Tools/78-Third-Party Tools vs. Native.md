Pinterest 有官方合作伙伴。

其他工具，如 Tailwind，使用 Pinterest 的 API 进行每次 Pin 上传，这有时可能更快被标记为垃圾邮件。Pinterest 更倾向于通过其原生 Scheduler 或批准的合作伙伴上传的 Pin。

根据我的经验，原生上传的 Pin 往往比通过外部工具发布的 Pin 表现稍好。然而，很难精确测量这一点。

为了公平比较，你需要在具有相同年龄、优化和现有 Pin 数量的账户上测试相同的 Pin —— 这并不容易。即使如此，Pinterest 也不喜欢重复的 Pin，并且可以识别不同账户上的相同图像，这使得结果更难比较。

我个人使用 Tailwind 了一段时间，然后切换回 Pinterest 的原生 Scheduler。原生上传获得展示的速度更快，整体表现也稍好一些。

虽然差异不是很大，但我仍然推荐使用原生 Scheduler 以获得最佳效果！
