这个版本的功能

这个 Automation：

1. 遍历你 Pinterest 账户中的所有 Board
2. 从每个 Board 获取最新的 250 个 Pin
3. 筛选符合条件的 Pin：
   - 发布超过 90 天 并且
   - 最近 90 天内少于 10 次查看
4. 然后删除它们。立即。毫不留情。
5. 错误处理（如果 Pinterest API 触发太快，会暂停 100 秒）

### 如何设置和使用这个 Automation？

下载以下场景 blueprint：

The Quick Fix – Simple Version (Limited to 250 pins - board).blueprint.json(11.9 KB)

按照"3. How to Set It Up in make.com"章节中的说明，将场景导入 make.com。

### 这个版本的优点

- 清理所有 Board
- 设置和运行快速
- 低复杂度，高效果

### 限制

- 每个 Board 只能访问最近的 250 个 Pin
- 不支持分页或深度清理
- 大型 Board 中超过 250 的旧 Pin 无法处理

### 最佳使用场景

- 拥有中小型 Board 的账户
- 快速月度维护
- 想要结果而不想深入了解高级过滤器或逻辑的用户

### 想要更改过滤逻辑？

你完全可以自定义删除条件：展示数、点击数、保存数，甚至关键词。

但要小心：过滤器控制一切。

在第 4 章中学习如何安全编辑它们：Handle With Care – Advanced Filtering / 小心操作 – 高级过滤
