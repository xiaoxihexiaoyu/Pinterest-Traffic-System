## 第 1 页

PSS 旨在为这个业务中每项（重要）任务提供标准操作程序（SOP），你可以在这里找到它们。
你会找到每个步骤的独立 SOP，以及包含所有 SOP 的汇编版本。这些 SOP 旨在辅助你的工作；例如，你可以在单独的屏幕上打开它们，在执行这些任务时，以确保你以最高效的方式完成任务。
或者
你可以将个别任务甚至整个策略委派给 VA（虚拟助理）。
所有其他模块的作用是更详细地解释这些工作流程和任务。它们为你提供重要的背景信息，并帮助你建立适合你每周排程的高效工作流程。
此外，它们还帮助你设置工具，并展示如何最好地集成它们。
如果你不知道从哪里开始，从这里开始：
PinScaleSystem 的结构让你在开始时就能看到"核心任务路线图"。
这些是核心任务——推动 80% 结果的最重要行动。
对于每一项，你都会获得我们确切的分步工作流程（SOP）。
我建议逐一浏览它们，然后将这些工作流程付诸实践。作为额外的关键模块，我建议：
- 基础设置（这能帮助你一劳永逸地妥善组织）
- 关于 Pinterest 算法（清晰解析算法的实际运作方式）
这些真的是你开始所需的全部：
1. 完成基础设置（使用 Overview Sheet + Google Drive 进行高效组织）
2. 使用 SOP 完成核心任务
3. 如果你愿意，还可以阅读关于 Pinterest 算法的内容
仅凭这三个步骤，你就能很好地做好快速有效开始的准备。
所有其他模块都更深入，并更详细地解释流程，但你不需要立即使用它们——你可以稍后逐步浏览。
你会在 Pin Tool Kit 和 Additional Tools & Automation 部分找到所有工具。
