## 第 1 页

你可以直接在 Pintoolkit.com 访问我们的工具。
为了确保只有 PinScaleSystem 的用户可以注册，请使用你在此注册时使用的相同电子邮件地址。
首次登录时，请重置你的密码。电子邮件可能需要几分钟才能到达，如果你在收件箱中没有看到它，也请检查你的垃圾邮件文件夹。
你可以轻松地使用你的 OpenAI API 使用我们的工具，因此你不需要支付任何额外费用来使用它们，除了少量的 OpenAI 额度。
你可以在这里设置 API key：https://platform.openai.com/api-keys
