# 第四步：创建 Pin 和 Board 描述

## 内容

Make it yours: https://docs.google.com/document/d/1DfK8XGYonzbFia9r-frx8eB6Bm9-s5Z6ccaDVEyVtW0/copy

1. 从总览表中获取关键词。
2. 前往 Pinclicks* > Top Pins > 输入你的关键词 > 导出 > Annotated Interests（注释兴趣）。
3. 打开 Google Sheets。
4. 前往 File > Open > Upload 并上传注释 CSV 文件。
5. 选择 B 列 > 右键单击 > 选择"Sort sheet Z to A"（Z 到 A 排序）。

### 第二页

6. 选择 4 个相关注释（"Occurrences"——数字越高表示相关性越高，优先选择这些）。
7. 使用 ChatGPT 创建描述，或使用我们的 Pin & Board Description Generator（Pin 和 Board 描述生成器）自动完成此步骤。
8. 将完成的描述添加到你的总览表中。

#### Pin 描述提示词

请为我的 Pin 创建一个个性化、听起来像真人撰写的 Pinterest 描述（500 个字符以内），包含此关键词：[插入关键词]。

在末尾包含一个简短的 CTA（行动号召），如"Read Now"（立即阅读）或"Save for Later"（稍后保存）。[插入 4 个注释]

#### Board 描述提示词

请创建一个个性化、听起来像真人撰写的 Pinterest Board 描述，包含这些关键词。

保持在 300 个字符以内，并使用完全匹配的关键词。[插入你的关键词和 4 个注释]

或者你可以使用我们的 Pin & Board Description Generator 自动完成此步骤。
