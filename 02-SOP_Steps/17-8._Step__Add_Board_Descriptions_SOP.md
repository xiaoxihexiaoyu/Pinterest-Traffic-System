# 第八步：添加 Board 描述

## 内容

这个过程非常简单。但由于在每次上传时直接添加它们总是有点烦人，我每周添加一次（通常在周日）。

Make it yours: https://docs.google.com/document/d/1bcfBWvy0XsjtW-qQBTqLlVBtfBHQ6gp_tQ5uVUZhBNc/copy

### 添加 Board 描述 SOP

1. 打开你的 Pinterest Board。
2. 点击右上角的三个点 > edit board（编辑 Board）。
3. 从你的总览表中插入你的 Board 描述。
