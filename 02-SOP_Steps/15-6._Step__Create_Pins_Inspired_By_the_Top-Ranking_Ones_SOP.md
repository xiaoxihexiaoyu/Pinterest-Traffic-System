# 第六步：创建灵感 Pins

## 内容

Make it yours:
https://docs.google.com/document/d/1pTz4s3otqTcDMgTX6bUfcKAuc9xnhvzdy1tAPhAIThA/copy

### 从总览表中为每个关键词高效创建新图片的步骤

1. 使用我们的 AI Image Prompt Generator（AI 图片提示词生成器）生成 Midjourney 提示词，并将其保存到 Google Spreadsheet 中，每个关键词一个标签页。
2. 使用我们的 Image Prompt Automation（图片提示词自动化）生成 AI 图片。
3. 从 Midjourney 下载生成的图片。
4. 解压 zip 文件并将图片组织到文件夹中。
5. 使用 xnConvert 对图片进行后处理。
6. 随机打乱和重命名文件。
7. 将图片上传到 Google Drive。

### 第一步：生成 Midjourney 提示词

在隐身标签页中打开 Pinterest。

输入你想生成图片的关键词。

### 第二页

下载排名前 10 的 3 张图片。

打开 AI Prompt Generator，如果尚未完成则配置你的 API 密钥，然后将 3 张图片上传到生成器中。

点击"Generate Prompts"开始生成提示词。不到一分钟，你将收到 10 个像这样的提示词。

![15-6._Step__Create_Pins_Inspired_By_the_Top-Ranking_Ones_SOP_page2_img0.png](assets/02-SOP_Steps/15-6._Step__Create_Pins_Inspired_By_the_Top-Ranking_Ones_SOP_page2_img0.png)

### 第三页

点击"Copy All"复制生成的提示词。

打开现有的或新的 Google Spreadsheet。

将复制的提示词粘贴到电子表格中，并重命名标签页以便更好地组织。

### 第四页

尽可能为更多关键词生成更多提示词，以使接下来的步骤更加高效！

### 第二步：使用我们的 Image Prompt Automation 生成图片

注意：我强烈建议在第二台电脑上或在你不需要电脑时运行自动化——例如，在睡觉前启动它。

在自动化运行时，你的电脑将被锁定，无法用于其他任务。

在第二个浏览器窗口中打开 midjourney.com 并前往 Create（提示：你也可以对 ideogram.ai 执行类似操作，特别是对于文字叠加 Pins）。

![15-6._Step__Create_Pins_Inspired_By_the_Top-Ranking_Ones_SOP_page4_img0.png](assets/02-SOP_Steps/15-6._Step__Create_Pins_Inspired_By_the_Top-Ranking_Ones_SOP_page4_img0.png)


![15-6._Step__Create_Pins_Inspired_By_the_Top-Ranking_Ones_SOP_page4_img1.png](assets/02-SOP_Steps/15-6._Step__Create_Pins_Inspired_By_the_Top-Ranking_Ones_SOP_page4_img1.png)


### 第五页

下载 Image Prompt Automation，启动它，并根据你的需求进行配置：

- 关键词数量 = 表中的标签页数量
- 每个关键词的提示词数 = 每个关键词有多少个提示词（我们通常使用 10 个）
- 每个提示词的生成数 = 每个提示词应生成多少张图片

确保带有 Midjourney 的浏览器窗口和带有 Google Spreadsheet 的浏览器窗口是你打开的最后两个窗口，其中 Google Spreadsheet 是最后一个。

然后，切换回 Image Prompt Automation 工具并点击 Start。

重要提示：工具运行时不要触摸键盘或鼠标！

该工具将最小化，并通过模拟键盘和鼠标操作自动将每个提示词从 Google Spreadsheet 复制到 Midjourney 中。

Image Prompt Automation 完成后，你将收到一条简短的消息。此时起，你可以再次使用鼠标和键盘。

![15-6._Step__Create_Pins_Inspired_By_the_Top-Ranking_Ones_SOP_page5_img0.png](assets/02-SOP_Steps/15-6._Step__Create_Pins_Inspired_By_the_Top-Ranking_Ones_SOP_page5_img0.png)


### 第六页

如果你是 Midjourney 或 Ideogram 的新手，请查看我们的 Midjourney SOP 或 Ideogram SOP 获取详细指南。

### 第三步：从 Midjourney 下载图片

专业提示：按关键词批量下载图片。这使得后续处理更加容易，因为你不必稍后再进行排序（见第四步）。

**如果你使用 Midjourney：**

前往"Organize"并滚动到你最近生成的图片。

通过按住鼠标左键或在点击每张图片时按住 SHIFT 来选择所有要下载的图片。

![15-6._Step__Create_Pins_Inspired_By_the_Top-Ranking_Ones_SOP_page6_img0.png](assets/02-SOP_Steps/15-6._Step__Create_Pins_Inspired_By_the_Top-Ranking_Ones_SOP_page6_img0.png)

### 第七页

完成后，点击"Download"，所有图片将被打包成 ZIP 文件（每个文件 50 张图片）。

**如果你使用 Ideogram：**

前往"Creations"并滚动到你最近生成的图片。

图片按四张一组分组，因此每组你只需要选择一张图片。

要下载多张图片，左键单击每组中一张图片上的白色复选框。

或者，在图片下方左键单击，按住并拖动以一次选择多张图片。

完成后，点击"Download"，所有图片将被打包成 ZIP 文件（每个文件最多 50 张图片）。

将下载的 ZIP 文件移动到你要处理它们的指定文件夹——我们建议使用 C:/Temp 以便于访问。

### 第八页

### 第四步：解压你的图片并将它们分类到文件夹中

使用你选择的 ZIP 工具解压所有 .zip 文件，或使用我们的批处理文件进行批量解压以节省时间：

1. 下载批处理文件"rename_and_unpack.bat"
2. 将其复制到存储所有 ZIP 文件的同一文件夹中（例如，C:/Temp）
3. 运行批处理文件，第一次可能会收到警告，点击"More info"。

![15-6._Step__Create_Pins_Inspired_By_the_Top-Ranking_Ones_SOP_page8_img0.png](assets/02-SOP_Steps/15-6._Step__Create_Pins_Inspired_By_the_Top-Ranking_Ones_SOP_page8_img0.png)

### 第九页

4. 点击"Run anyway"（仍要运行）
5. 批处理文件将自动重命名所有 .ZIP 文件，从文件名中删除特殊字符（以防止任何问题），然后解压所有 ZIP 文件。
6. 过程完成后，你将在命令行窗口中看到确认消息。

![15-6._Step__Create_Pins_Inspired_By_the_Top-Ranking_Ones_SOP_page9_img0.png](assets/02-SOP_Steps/15-6._Step__Create_Pins_Inspired_By_the_Top-Ranking_Ones_SOP_page9_img0.png)

7. 你现在可以删除所有 ZIP 文件和"rename_and_unpack.bat"文件，因为接下来的步骤不再需要它们。

如果你以后需要它们，可以在回收站中找到。

如果需要，将图片分类到以总览表中相应关键词命名的文件夹中，以便于处理。

xnConvert 现在将把所有图片转换为新文件，同时去除所有不必要的元数据。

### 第六步：随机打乱和重命名文件

为了防止太多相似的图片按顺序出现，最好在每个关键词文件夹内随机打乱图片。此外，文件名需要重命名，因为它们包含不必要的详细信息，如 Midjourney 用户名和提示词。

### 第十四页

#### 此脚本的作用

- 它进入每个子文件夹。
- 它随机打乱内部的所有图片。
- 它根据子文件夹名称（= 关键词）重命名它们。
- 它重新编号它们以获得更自然、整洁的结构。

#### 如何使用

1. 下载批处理文件"shuffle_and_rename_files_in_subfolders.bat"。
2. 将文件复制到所有 ZIP 文件所在的同一文件夹中（例如，C:/Temp）。
3. 运行批处理文件。
4. 如果第一次收到安全警告，点击"More info"并继续。

### 第七步：将图片上传到 Google Drive

不要逐张将完成的图片直接上传到 Pinterest，而是先将它们全部上传到 Google Drive，这样效率更高。这样，你和你的团队可以将所有图片存储在一个中央位置，便于访问。

下载并安装 Google Drive for Desktop 并将其连接到你的 Google 账户。

安装后，你的电脑上将出现一个新驱动器（通常是 G:）。
