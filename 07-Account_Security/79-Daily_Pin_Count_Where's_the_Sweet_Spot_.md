## 内容

去年，我们进行了广泛的测试，甚至将一些账户推向每天 100-200 个 Pin（200 是 Pinterest 的官方每日限制）。
**最佳表现**始终保持在每天 24-48 个 Pin 之间。然而，在过去 2-3 个月里，我们注意到一个明显的转变——现在更少的 Pin（每天 8-12 个）表现最佳。

**为什么？**
Pinterest 优先考虑保存和互动率。如果在一天内上传了太多 Pin，该平台将难以通过注释、搜索结果和主页信息流正确地对它们进行分类和分发。
这会**损害你的整体账户表现**，并导致**恶性循环**，使新 Pin 难以获得流量。
你可以在此处了解我们的**主要实验以及从中获得的经验**。
专注于：
1. **少即是多**：
具有**强劲保存率和互动率**的高质量 Pin 优于大量上传。
2. **定期修剪表现不佳的内容（Pin 和 Board）**：
太多低互动内容会浪费资源，并会**降低你的整体覆盖面**。
3. 几个月前，**所有 Pin** 都会出现在关注者的主页信息流中，无论你上传多少
这自然提高了你的互动率。
现在，**Pinterest 选择性地展示内容**，使良好的质量和互动信号比以往任何时候都更重要。

### 我推荐新账户使用此策略：

从每天 5-10 个 Pin 开始，逐渐增加到每天 8-12 个。
一旦你的互动率和保存率强劲，就可以测试扩展到每天 20-25 个 Pin。

### 查看你的表现：

如果你的保存量和曝光量保持高位，你可以保持每天 20-25 个 Pin 或进一步扩展。
但是...
如果你的许多新 Pin 即使几天后仍收到 **<10 次曝光**，请将 Pin 数量减少到每天 8-12 个，并**发布新的 Scout Pin** 以提升你的整体互动。
