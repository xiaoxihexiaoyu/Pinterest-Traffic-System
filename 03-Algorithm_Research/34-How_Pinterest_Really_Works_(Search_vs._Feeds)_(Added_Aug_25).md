# Pinterest 真正的工作原理

## 内容

### 第一页

大多数创作者只谈论 Search 排名。是的，Search 很重要。但问题是：Pinterest 的大部分流量甚至不是来自 Search。

它来自 Home Feed + Recommendation Feed。

所以，让我们来分解一下：

### 1. Search = 关键词驱动

当有人输入查询时，Pinterest 会匹配针对以下内容优化的 Pin：

- **Annotations**（Pinterest 的内部标签——这里有一份分步 SOP：Create Pin Board Descriptions with Annotations）
- **Keyword targeting**（标题 + 描述）
- **Clean metadata**（您的 URL + 页面标题/描述（基本页面 SEO）应该匹配）
- **Text relevance**（您的链接页面与 Pin 高度相关）

如果您掌握这些，当有人主动搜索时，您就会出现。

### 2. Home Feed = 意图驱动

这是一个完全不同的游戏。在这里，Pinterest 在用户甚至输入任何内容之前就预测他们可能想要什么。

#### 如何？通过：

- 将 Pin 聚类到 Interests 中（例如，"fall home decor"）
- 将它们与 Taxonomy 进行映射（Pinterest 的结构化兴趣数据库）
- Entity mapping（将您的 Pin 与识别的主题/实体联系起来）
- Board graph distribution（Pinterest 通过相关的 Board + 推荐引擎传播您的 Pin）

翻译过来就是：您不是在竞争一个 Keyword。您是在竞争整个用户旅程。
