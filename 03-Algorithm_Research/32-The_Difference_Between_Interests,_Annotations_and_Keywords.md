## 内容

### 第一页

### 1. Interests vs. Annotations：兴趣与标注

**Interests（兴趣）** 是 Pinterest 使用的大主题（如"DIY Crafts"或"Home Decor"）。

它们是广泛的类别，有助于将类似的 Pin 分组在一起。

**Annotations（标注）** 是 Pinterest 根据图像和文本自动为您的 Pin 添加的特定标签。

它们有助于确定您的 Pin 适合哪个兴趣。

### 2. Interests vs. Keywords：兴趣与关键词

**Keywords（关键词）** 是人们在 Pinterest 搜索时输入的确切词语（如"DIY project ideas"）。

**Interests** 是更广泛的主题，包含许多相关的 Keywords。它们帮助 Pinterest 理解整体主题，即使搜索不使用确切的词语。

### 3. Keywords 到底是什么，为什么需要 Interests？

Keywords 精确地描述您的 Pin 的内容。

### 第二页

**Interests** 的存在使 Pinterest 能够将许多相关的 Keywords 分组到一个大类别中，从而更容易将您的 Pin 与用户的喜好相匹配。

### 4. Taxonomy 和 Interest Mapping：分类法和兴趣映射

**Taxonomy（分类法）** 就像兴趣的家谱——它将广泛的主题组织成更小、更具体的主题（例如，"Food" → "Dinner" → "Lasagne"）。

**Interest Mapping（兴趣映射）** 是将您的 Pin 中的 Keywords 和 Annotations 链接到这个分类法的正确部分的过程，以便 Pinterest 可以向合适的人展示它。

简而言之，您使用 Keywords 来描述您的 Pin，Annotations 帮助 Pinterest 正确标记您的 Pin，而 Interests（由 Taxonomy 组织）将这些标签分组为大主题，以便更好地与用户搜索匹配。

#### 为什么 Keyword 研究和 Annotations 如此重要

**精确性**：在您的描述中使用经过研究的 Keywords 和清晰的 Annotations 有助于 Pinterest 准确理解您的 Pin 的内容。

**更强的信号**：当您包含正确的 KW 和 Annotations 时，您的 Pin 会向 Pinterest 的算法（如 PinSage）发送更强的信号。这提高了其"嵌入相似性"——这是一种技术性的说法，表示它更可能与用户兴趣匹配并出现在 Feed 中。

**优化**：没有这些清晰的 KW 和 Annotations，您的 Pin 可能会匹配得不那么精确，甚至错误，尽管 Pinterest 仍然会做出很好的猜测。
